# Seja bem vindo ao meu projeto em Java!

Este projeto documenta meu aprendizado em desenvolvimento full stack, mais específicamente utilizando a linguagem **Java**. Nele eu vou tentar colocar tudo que eu aprendi em prática, unindo as coisas mais importantes. Vale ressaltar que não houve uso de IA para a geração do código.

# Objetivo

Meu objetivo com este projeto é utiliza-lo como um laboratório pessoal de estudos, com foco em aprender e documentar minha evolução na programação, deixando minha lógica de programação cada vez melhor e o código cada vez mais limpo, e claro, aprender cada vez mais tecnologias e conceitos.

## Atualmente

Atualmente o projeto conta apenas com a arquitetura MVC (Model, View e Controller), operações matemáticas, conceitos de POO aplicados, e versionamento e controle de versão por meio do GitHub.

## Futuramente

Futuramente eu pretendo implementar muitas novidades, conceitos como tratamento de exceções (try/catch), autenticação de usuário via token, Clean Code, Continous Integration e Continous Delivery (CI/CD) implementados por meio de DevOps, Design Patterns, API Rest para integrar back-end com front-end, adicionar o framework SpringBoot, usar funções Lambda e programação funcional, além de banco de dados e RabbitMQ em container.
